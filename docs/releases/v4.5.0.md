# EasyCord v4.5.0 — Stable Release
**Release date:** 2026‑05‑02  
**Status:** Stable — Production‑Ready

v4.5.0 promotes the localization hardening line validated in v4.5.0‑beta.3.  
This is a validation‑only promotion — **no new features** were added after the release candidate.

## Highlights
- Production‑grade localization fallback chains with deterministic, bounded resolution
- STRICT diagnostics correctness: missing keys now always raise in STRICT mode
- Safe metrics snapshots from `get_metrics()` with isolated locale‑frequency data
- BCP‑47 locale validation including script subtags (e.g. `zh‑Hant‑HK`, `sr‑Latn‑RS`)
- Performance regression enforcement in GitHub Actions
- Release consistency checks for version pins, release notes, and built‑artifact hygiene
- Clean wheel and source‑distribution packaging

## Installation

```bash
pip install "easycord @ git+https://github.com/rolling-codes/EasyCord.git@v4.5.0"
```

Or from source:

```bash
git clone --branch v4.5.0 https://github.com/rolling-codes/EasyCord.git
cd EasyCord
pip install -e ".[dev]"
```
